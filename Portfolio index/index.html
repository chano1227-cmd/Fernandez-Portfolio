My Projects

Punada Martial Arts website

This front-end project is a basic informational website created for Punada Martial Arts, serving as its essential online presence. The site was developed to offer students and potential members quick access to core information, including class schedules, contact details, and recent news and updates. It is designed to be a clean, central hub for all basic necessities related to the martial arts center.

CraftyHive Website

I developed the front-end for a simple e-commerce website selling flower bouquets. The core of this project involved implementing essential shopping functionality, specifically the product listing component and the add-to-cart logic. I focused on writing clean code to manage the state of the shopping cart and dynamically render product data. The final interface was styled to be visually inviting and user-friendly, serving as a practical exercise in combining basic front-end logic with a welcoming, aesthetically pleasing layout.

Data Analytics Project

Analytics Showcase
